# Что такое RAG

RAG — это метод поиска и генерации ответов. Сначала находятся похожие фрагменты текста, потом они передаются в модель для ответа.

Этапы:
1. Текст режется на куски.
2. Каждый кусок превращается в числа.
3. Вопрос тоже превращается в числа.
4. Находятся самые похожие куски.
5. Они передаются в модель, которая пишет ответ.

Главное правило — модель должна отвечать только по тем кускам, которые нашла. Нельзя выдумывать.
