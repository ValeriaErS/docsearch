# Что такое векторное хранилище

Векторное хранилище — это база данных, где хранятся числа, полученные из текста.

Векторы можно хранить по-разному. Простой способ — в памяти, в списке. Так делают для маленьких проектов.

Для больших проектов используют специальные базы данных, которые ищут быстрее.

Поиск работает по косинусной близости — это способ сравнить два вектора и понять, похожи ли они.
