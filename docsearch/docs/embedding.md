# Что такое эмбеддинги

Эмбеддинг — это способ представить текст в виде чисел. Вместо слов используются списки чисел.

Похожие по смыслу слова имеют похожие наборы чисел. Например, слова "собака" и "пёс" будут близки, а "собака" и "компьютер" — далеки.

Эмбеддинги нужны для поиска по смыслу. Машина не понимает слова, но понимает числа.

Эмбеддинги можно получить через нейросети. Они обучаются на больших текстах и могут превращать любые слова в числа.
